# 🌎 Brasil Encantado – Portal Turístico Interativo

O **Brasil Encantado** é um portal turístico desenvolvido com o objetivo de apresentar os principais destinos do Brasil de forma organizada, visualmente atrativa e de fácil navegação.

## 📌 Sobre o Projeto
Este projeto foi criado como parte de um trabalho acadêmico, com foco na aplicação de:
- HTML5 semântico
- CSS externo
- Estruturação de páginas web
- Navegação intuitiva

A proposta é simular um site de uma agência de turismo fictícia, oferecendo informações sobre destinos, cultura, gastronomia e experiências pelo Brasil.

## 🎯 Objetivos
- Divulgar destinos turísticos brasileiros
- Criar uma navegação clara e intuitiva
- Aplicar boas práticas de desenvolvimento web
- Desenvolver um site responsivo (nível básico)

## 🧩 Funcionalidades
- Página inicial com destaques
- Páginas de destinos turísticos
- Seção de cultura e gastronomia
- Navegação entre páginas interligadas
- Uso de imagens e elementos multimídia

## 🛠️ Tecnologias Utilizadas
- HTML5
- CSS3

## 📱 Responsividade
O site possui responsividade básica, adaptando-se a diferentes tamanhos de tela.
